---
title: 概述
---

服务器提供了通过 MCP 为语言模型添加上下文的基础构建块。这些原语实现了客户端、服务器和语言模型之间的丰富交互：

- **Prompts（提示）**：指导语言模型交互的预定义模板或指令
- **Resources（资源）**：为模型提供额外上下文的结构化数据或内容
- **Tools（工具）**：允许模型执行操作或检索信息的可执行函数

每个原语可以总结为以下控制层次结构：

| 原语      | 控制方       | 描述                           | 示例                    |
| --------- | ------------ | ------------------------------ | ----------------------- |
| Prompts   | 用户控制     | 由用户选择触发的交互式模板     | 斜杠命令、菜单选项      |
| Resources | 应用程序控制 | 由客户端附加和管理的上下文数据 | 文件内容、git 历史      |
| Tools     | 模型控制     | 向 LLM 暴露以执行操作的函数    | API POST 请求、文件写入 |
