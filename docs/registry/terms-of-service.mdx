## 概述

这些条款（"条款"）管辖您对官方 MCP 注册表（托管在 [https://registry.modelcontextprotocol.io/](https://registry.modelcontextprotocol.io/) 或后继位置的服务）（"注册表"）的访问和使用，包括 MCP 服务器的提交或发布、对 MCP 服务器或关于此类服务器和/或其开发者的数据的引用（"注册表数据"）以及相关行为。注册表旨在成为社区成员开发的 MCP 服务器的集中式仓库，以方便 AI 应用程序访问。

这些条款受加利福尼亚州法律管辖。
