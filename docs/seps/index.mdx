## SEP 状态定义

| 状态                                              | 定义                                      |
| ------------------------------------------------- | ----------------------------------------- |
| <Badge color="gray" shape="pill">草稿</Badge>     | 已有赞助者的 SEP 提案，正在进行非正式评审 |
| <Badge color="yellow" shape="pill">评审中</Badge> | SEP 提案已准备好由核心维护者进行正式评审  |
| <Badge color="blue" shape="pill">已接受</Badge>   | SEP 已接受，等待参考实现                  |
| <Badge color="green" shape="pill">最终</Badge>    | SEP 已完成定稿，参考实现也已就绪          |
| <Badge color="red" shape="pill">已拒绝</Badge>    | SEP 被核心维护者拒绝                      |
| <Badge color="red" shape="pill">已撤回</Badge>    | SEP 被作者撤回                            |
| <Badge color="purple" shape="pill">已取代</Badge> | SEP 已被更新的 SEP 取代                   |
| <Badge color="orange" shape="pill">休眠</Badge>   | SEP 无赞助者，6 个月后关闭                |
