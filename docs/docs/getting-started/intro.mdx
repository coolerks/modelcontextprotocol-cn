title: 什么是 Model Context Protocol (MCP)？
sidebarTitle: 什么是 MCP？
---

MCP（Model Context Protocol）是一种开源标准，用于将 AI 应用程序连接到外部系统。

通过 MCP，Claude 或 ChatGPT 等 AI 应用程序可以连接到数据源（如本地文件、数据库）、工具（如搜索引擎、计算器）和工作流（如专门提示词），从而访问关键信息并执行任务。

可以把 MCP 想象成 AI 应用程序的 USB-C 接口。正如 USB-C 提供了连接电子设备的标准化方式，MCP 提供了将 AI 应用程序连接到外部系统的标准化方式。

## MCP 能实现什么？

- Agent 可以访问你的 Google Calendar 和 Notion，充当更加个性化的 AI 助手。
- Claude Code 可以使用 Figma 设计生成整个 Web 应用程序。
- 企业聊天机器人可以连接到组织内的多个数据库，使用户能够通过聊天分析数据。
- AI 模型可以在 Blender 上创建 3D 设计并使用 3D 打印机将其打印出来。

## 为什么 MCP 很重要？

取决于你在生态系统中的位置，MCP 可以带来一系列好处。

- **开发者**：MCP 减少了构建或集成 AI 应用程序或 agent 时的开发时间和复杂性。
- **AI 应用程序或 agent**：MCP 提供了对数据源、工具和应用程序生态系统的访问，这将增强能力并改善最终用户体验。
- **最终用户**：MCP 带来更强大的 AI 应用程序或 agent，能够在必要时访问你的数据并代表你执行操作。

## 广泛的生态系统支持

MCP 是一个开放协议，受到广泛的 clients 和 servers 支持。[Claude](https://claude.com/docs/connectors/building) 和 [ChatGPT](https://developers.openai.com/api/docs/mcp/) 等 AI 助手，[Visual Studio Code](https://code.visualstudio.com/docs/copilot/chat/mcp-servers)、[Cursor](https://cursor.com/docs/context/mcp)、[MCPJam](https://docs.mcpjam.com/getting-started) 等开发工具，以及许多其他工具都支持 MCP —— 只需构建一次即可随处集成。
