## 开始使用

每个 SDK 提供相同的功能，但遵循其语言的习语和最佳实践。所有 SDK 都支持：

- 创建暴露 tools、resources 和 prompts 的 MCP server
- 构建可连接到任何 MCP server 的 MCP client
- 本地和远程传输协议
- 具有类型安全的协议合规性

访问你选择语言的 SDK 页面以查找安装说明、文档和示例。
