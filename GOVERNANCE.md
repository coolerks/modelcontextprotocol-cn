通用项目政策

Model Context Protocol 已设立为 LF Projects, LLC 的一个系列。适用于 Model Context Protocol 及 Model Context Protocol 参与者的政策，包括商标使用指南，位于 [https://www.lfprojects.org/policies/](https://www.lfprojects.org/policies/)。根据本治理文件的规定批准的治理变更也必须获得 LF Projects, LLC 的批准。

Model Context Protocol 参与者确认，所有新贡献的版权将由版权持有人保留为独立著作作品，并且任何贡献者或版权持有人均不需要将版权转让给本项目。

除非下文另有说明，所有对本项目的代码和规范贡献必须使用 Apache License 2.0 版本（此处获取：[https://www.apache.org/licenses/LICENSE-2.0](https://www.apache.org/licenses/LICENSE-2.0)）（"项目许可证"）。

所有对外发布的代码和规范将在项目许可证下提供。Core Maintainers 可以例外批准对入站或出站贡献使用替代开源许可证。

所有文档（不包括规范）将在 Creative Commons Attribution 4.0 International 许可证下提供，地址为：https://creativecommons.org/licenses/by/4.0。
